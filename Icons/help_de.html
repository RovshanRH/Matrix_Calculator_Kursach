Hilfebereich

Der Matrixrechner ist eine Anwendung zur Durchführung grundlegender arithmetischer Operationen mit zwei Matrizen. Die Anwendung verfügt über zwei Matrizen, die als "Matrix A" und "Matrix B" bezeichnet werden, sowie eine "Ergebnismatrix", die das Ergebnis der auf den Matrizen durchgeführten Operationen anzeigt.

Matrixbezogene Funktionen sind in die folgenden Abschnitte unterteilt:

  • Matrix
    • Matrixgröße
    • Matrixoperationen
    • Matrix-Kontextmenü
    • Matrixrang
  • Operationen
  • Kopfzeile
  • Anwendungs-Kontextmenü

Matrixbereich

Die zwei sequentiellen Operanden, Matrix A und Matrix B, werden als Tabellen mit einer Standardgröße von 3x3 dargestellt. Matrix A und Matrix B sind funktional identisch.

Matrixgröße

Um die Größe einer Matrix zu ändern, werden zwei Zähler von 1 bis 10 bereitgestellt. Der erste Zähler steuert die Anzahl der Zeilen, der zweite die Anzahl der Spalten. Um die Größenänderung anzuwenden, klicken Sie auf die Schaltfläche, die den Zählern folgt.

Matrixoperationen

Jede Matrix unterstützt vier Operationen: Matrixtransposition, Finden der inversen Matrix, Division durch eine Konstante und Multiplikation mit einer Konstante. Ein Eingabefeld const wird unter der Matrixtabelle bereitgestellt, um die Konstante anzugeben.

  • Transposition - Führt die Matrixtranspositionsoperation durch.
  • Finden der Inversen Matrix - Berechnet die inverse Matrix mit einer Fehlermarge von etwa 0,001–0,1 für Zellwerte. Das Programm gibt einen Fehler aus, wenn die Matrix singulär (entartet) ist.
  • Division durch eine Konstante - Jede Zelle wird durch die angegebene Zahl dividiert.
  • Multiplikation mit einer Konstante - Jede Zelle wird mit der angegebenen Zahl multipliziert.

Kontextmenü

Ein Rechtsklick auf eine Matrix zeigt ein Kontextmenü mit den folgenden Optionen an:

  • Kopieren - Kopiert die Matrixwerte in die Zwischenablage im TSV-Format.
  • Einfügen - Ersetzt die Matrixwerte mit Daten aus der Zwischenablage. Unterstützt TSV- und CSV-Formate.
  • Löschen - Ersetzt alle Matrixwerte durch Null.
  • Einfügen einer Einheitsmatrix - Ersetzt die Matrixwerte durch die einer Einheitsmatrix (Hauptdiagonale ist "1", alle anderen Zellen sind "0").
  • Einfügen einer oberen Dreiecksmatrix - Ersetzt die Matrixwerte durch die einer oberen Dreiecksmatrix (alle Zellen oberhalb oder auf der Hauptdiagonale sind "1", andere sind "0").
  • Einfügen einer unteren Dreiecksmatrix - Ersetzt die Matrixwerte durch die einer unteren Dreiecksmatrix (alle Zellen unterhalb oder auf der Hauptdiagonale sind "1", andere sind "0").

Matrixrang

Unter der Matrixtabelle befindet sich ein Rangzähler, der nach dem Klicken auf die Schaltfläche "Matrixrang finden" aktualisiert wird. Der Anfangswert ist "0".

Operationsbereich

  • Addition - Berechnet die Summe von Matrix A und Matrix B.
  • Subtraktion - Subtrahiert Matrix B von Matrix A.
  • Multiplikation - Multipliziert Matrix A mit Matrix B.

Das Programm verhindert, dass Benutzer nicht-quadratische Matrizen addieren oder subtrahieren oder Matrizen multiplizieren, die die Multiplikationsbedingungen nicht erfüllen.

  • Löschen - Ersetzt die Werte aller Matrizen durch Null.
  • Zufällige Werte einfügen - Fügt zufällige Werte in Matrix A und Matrix B ein.

Kopfzeilenbereich

Mit dem Zähler X können Sie eine Zahl angeben, die die Größe einer quadratischen Matrix für sowohl Matrix A als auch Matrix B festlegt. Um die Größenänderung anzuwenden, klicken Sie auf die Schaltfläche links vom Zähler.

Anwendungs-Kontextmenü

Ein Rechtsklick auf das Anwendungsfenster zeigt ein Kontextmenü mit den folgenden Optionen an:

  • 3 Matrizen kopieren - Kopiert die Werte aller drei Matrizen in die Zwischenablage im CSV-Format.
  • 3 Matrizen einfügen - Fügt Werte aus der Zwischenablage in Matrix A, Matrix B und Matrix C ein, jeweils im CSV- oder TSV-Format.
  • Matrizen löschen - Setzt alle Matrixwerte auf Null.
  • Matrizen tauschen - Tauscht die Positionen von Matrix A und Matrix B.
