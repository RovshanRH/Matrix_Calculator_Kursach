Help Section

The Matrix Calculator is an application for performing basic arithmetic operations on two matrices. The application features two matrices labeled "Matrix A" and "Matrix B", as well as a "Resulting Matrix" that displays the outcome of operations performed on the matrices.

Matrix-related functions are divided into the following sections:

  • Matrix
    • Matrix Size
    • Matrix Operations
    • Matrix Context Menu
    • Matrix Rank
  • Operations
  • Header
  • Application Context Menu

Matrix Section

The two sequential operands, Matrix A and Matrix B, are presented as tables with a default size of 3x3. Matrix A and Matrix B are functionally identical.

Matrix Size

To change the size of a matrix, two counters ranging from 1 to 10 are provided. The first counter controls the number of rows, and the second controls the number of columns. To apply the size change, click the button that follows the counters.

Matrix Operations

Each matrix supports four operations: matrix transposition, finding the inverse matrix, division by a constant, and multiplication by a constant. A const input field is provided below the matrix table to specify the constant.

  • Transposition - Performs the matrix transposition operation.
  • Finding the Inverse Matrix - Calculates the inverse matrix with an error margin of approximately 0.001â€“0.1 for cell values. The program returns an error if the matrix is singular (degenerate).
  • Division by a Constant - Each cell is divided by the specified number.
  • Multiplication by a Constant - Each cell is multiplied by the specified number.

Context Menu

Right-clicking on a matrix displays a context menu with the following options:

  • Copy - Copies the matrix values to the clipboard in TSV format.
  • Paste - Replaces the matrix values with data from the clipboard. Supports TSV and CSV formats.
  • Clear - Replaces all matrix values with zero.
  • Insert Identity Matrix - Replaces the matrix values with those of an identity matrix (main diagonal is "1", all other cells are "0").
  • Insert Upper Triangular Matrix - Replaces the matrix values with those of an upper triangular matrix (all cells above or on the main diagonal are "1", others are "0").
  • Insert Lower Triangular Matrix - Replaces the matrix values with those of a lower triangular matrix (all cells below or on the main diagonal are "1", others are "0").

Matrix Rank

Below the matrix table is a rank counter, which updates after clicking the "Find Matrix Rank" button. The initial value is "0".

Operations Section

  • Addition - Computes the sum of Matrix A and Matrix B.
  • Subtraction - Subtracts Matrix B from Matrix A.
  • Multiplication - Multiplies Matrix A by Matrix B.

The program prevents users from adding or subtracting non-square matrices or multiplying matrices that do not meet multiplication conditions.

  • Clear - Replaces the values of all matrices with zero.
  • Insert Random Values - Inserts random values into Matrix A and Matrix B.

Header Section

Using the X counter, you can specify a number that sets the size of a square matrix for both Matrix A and Matrix B. To apply the size change, click the button to the left of the counter.

Application Context Menu

Right-clicking on the application window displays a context menu with the following options:

  • Copy 3 Matrices - Copies the values of all three matrices to the clipboard in CSV format.
  • Paste 3 Matrices - Pastes values from the clipboard into Matrix A, Matrix B, and Matrix C, respectively, in either CSV or TSV format.
  • Clear Matrices - Sets all matrix values to zero.
  • Swap Matrices - Swaps the positions of Matrix A and Matrix B.
